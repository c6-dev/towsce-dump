# towsce-dump
 
SDK/MDK dump for The Outer Worlds: Spacer's Choice Edition.  

Created with [UEDumper](https://github.com/Spuckwaffel/UEDumper). Some structs needed to be changed (i.e. UObject has two extra bools).  

Offsets used (rva):  

* GNAMES: 0x6546700  
* GOBJECTS: 0x6582D70  
* GWORLD: 0x66CD6E0
